# Лабораторная работа №7 - To-Do приложение на Vue 3

## Выполнил Цвиль Павел ФИТ-231

##  Описание
Простое To-Do приложение с двумя страницами, написанное на Vue 3. 
Данные сохраняются в localStorage браузера.

##  Функционал
-  Добавление новых задач
-  Отметка выполнения (чекбокс)
-  Редактирование задач (двойной клик)
-  Удаление задач с подтверждением
-  Поиск по задачам
-  Две страницы: задачи и "О сайте"
-  Статистика выполнения
-  Сохранение в localStorage

##  Страницы
1. **Главная** - список задач
2. **О сайте** - статистика и информация об авторе

##  Технологии
- Vue 3 (Composition API)
- HTML5 / CSS3
- Font Awesome для иконок
- LocalStorage для хранения данных
